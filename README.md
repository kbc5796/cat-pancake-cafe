Cat Pancake Cafe:
A high-paced simulation game built in the Godot Engine. The project focuses on real-time state management and a modular "order-to-delivery" gameplay loop.

Features:
Gameplay Loop: Real-time state management between customer orders, machine interactions, and item fulfillment.

Progression System: Multi-level difficulty scaling and dynamic win/loss conditions implemented in GDScript.

Customization: Modular system for handling multiple character cosmetic states and sprites.

Asset Integration: Efficient implementation of high-fidelity art and localized audio, ensuring smooth performance and low load times.

Technical Stack: 
Engine: Godot

Language: GDScript

Version Control: Git
